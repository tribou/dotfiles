# Product

## What is this?
This is a personal dotfiles repository designed to bootstrap developer environments quickly, predictably, and with advanced CLI tooling, tmux workflows, and AI integration.

## Success Criteria
- Environments can be provisioned rapidly from scratch using `bootstrap.sh`.
- The CLI workflow is extremely fast, fuzzy-finding-heavy, and tmux-integrated.
- Developer experience is seamless across macOS/Linux, though macOS is the primary target.
