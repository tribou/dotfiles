# Example Skill

This is a placeholder skill. Replace this file with your own custom skill content.

Skills are markdown files with YAML frontmatter (`name`, `description`) followed by instructions.

## Format

- **name**: Skill identifier (lowercase, hyphens optional)
- **description**: One-line summary shown in the skills list

## Instructions

Write your skill's instructions in markdown below the frontmatter. These instructions guide Claude Code when the skill is invoked.

For detailed guidance on writing skills, use `superpowers:writing-skills`.
